ReadyState Description
0 (UNSENT) Request abhi banayi gayi hai, lekin open() method call nahi hua.
1 (OPENED) open() method call ho chuka hai.
2 (HEADERS_RECEIVED) Server se response headers mil gaye hain.
3 (LOADING) Response ka body part-by-part receive ho raha hai.
4 (DONE) Request pura ho gaya hai, aur response available hai.
